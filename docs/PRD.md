# PRD — AI Panel Studio

## 产品定位

AI Panel Studio 是一个基于大语言模型的多专家圆桌讨论模拟平台。用户输入话题后，系统自动生成主持人+专家嘉宾阵容，通过 AI 驱动每位嘉宾的独立发言决策，模拟真实的圆桌讨论互动，并实时提炼共识与分歧。

## 核心用户故事

1. **创建讨论**：用户输入话题，选择嘉宾人数（2-6人），系统调用 AI 生成阵容（包含姓名、头衔、立场）。
2. **沉浸式观看**：用户进入 Studio 演播室，观看 AI 嘉宾实时讨论。每位嘉宾有独立的思考气泡和状态指示灯。
3. **实时洞察**：系统自动转录全部发言，每隔 5 轮提炼共识点和分歧点。
4. **结束总结**：用户结束讨论后，系统生成自然语言总结。

## 非功能需求

- 多个讨论并行运行，数据完全隔离
- 前端响应式设计，支持桌面（宽屏）和移动端（窄屏堆叠）
- 后端 API 全 RESTful 风格
- WebSocket 实时推送，事件类型标准化
